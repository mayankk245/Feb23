Below we are using a custom pipe named as agePipe which is taking our date of birth date and return us the age.

Syntax: dob | agePipe : true : true, dob is the variable in our Ts file and after | symbol we are using our pipe name which is having 2 optional inputs:


After pipe name the inputs are seperated by ':' colon


  1. upperCase: Should convert result to upper case a boolean parameter.
  2. emoji: Should use emoji in the result.

Use Case:


  1. Not Passing any optional parameter: dob | agePipe

    2. I am "{{dob | agePipe }}"


  2. Passing upeerCase value as true: dob | agePipe : true

    3. I am "{{dob | agePipe : true}}"


  3. Passing both value Upper case and use emoji as true: dob | agePipe : true : true

    4. I am "{{dob | agePipe : true : true}}"

{{
"import { Pipe, PipeTransform } from '@angular/core';

@Pipe({
  name: 'agePipe'
})
export class AgePipePipe implements PipeTransform {

  transform(value: Date|null|undefined , upperCase: boolean = false, emoji:boolean = false): string|null|undefined {
    if (value == null || value == undefined) return value

    let text = ' years old '
    let currentDate = new Date().getFullYear();
    let age = currentDate - value.getFullYear();

    if(upperCase){
      text = text.toUpperCase();
    }
    if(emoji){
      text = text + "ðŸ¥³"
    }
    return age + text;
  }
   
} "
