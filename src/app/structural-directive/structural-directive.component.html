appAntiIf directive: It is a custom structural directive which works opposite to *ngIf. It is taking two inputs 1st one is mandatory and 2nd is optional:

Naming convention of structural direct inputs: 1st input which is by default by the name of directive will be taken as 1st input.
But all the other inputs which are going to be part of a structural directive the prefix of those inputs should contain the name 
of directive followed by the actual name of input.

Note: See the code for how to take input


  1. appAntiIf: It is name of the structural directive and should be use with * as prefix on the element Takes a boolean value as input.
  2. appAntiIfElseBlock: Optional input used on element to provide the else part of the condition. On element we should only use "elseBlock" to provide the input.

Whenever the input value will be true you will see this content. It is passed as a templet refrence to the elseBlock Input of custom directive. And it will render if 1st input value is true and a refrence of this element is passed to 2nd input which is elseBlock in this case.

Press toggle button to see the Explanation.

Code Example of structural directive


{{"
import { Directive, Input } from '@angular/core';
import { ViewContainerRef, TemplateRef } from '@angular/core';

@Directive({
  selector: '[appAntiIf]'
})
export class AntiIfDirective {
  private notShow:boolean = false
  constructor(private vc:ViewContainerRef, private tempRef: TemplateRef) { }
  

  @Input() set appAntiIf(val:boolean){
    this.notShow =  val;
    this.vc.clear();
    if(!this.notShow){
      this.vc.createEmbeddedView(this.tempRef);
    }
    else if(this.appAntiIfElseBlock){
      this.vc.createEmbeddedView(this.appAntiIfElseBlock);
    }
  }

  @Input() appAntiIfElseBlock?: TemplateRef;


  ngOnInit(){
    this.vc.clear();
    if(!this.notShow){
      this.vc.createEmbeddedView(this.tempRef);
    }
    else if(this.appAntiIfElseBlock){
      this.vc.createEmbeddedView(this.appAntiIfElseBlock);
    }
  
  }

}"}}
